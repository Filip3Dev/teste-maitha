# koa-api

Para iniciar o projeto, rode o comando:

``
npm install
``

Após isso, você deve renomear o arquivo `.env.example` para `.env`

Adicione as variaives de ambiente no arquivo `.env`

E então inicie o projeto com o comando

``
npm run dev
``
